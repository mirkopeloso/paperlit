## Installazione


su MYSQL:
CREATE DATABASE paperlit;
CREATE USER 'paperlit'@'localhost' IDENTIFIED WITH mysql_native_password BY 'my-strong-password-here';



composer install
php artisan migrate:fresh
php artisan breeze:install
sudo n stable
npm install && npm run dev
