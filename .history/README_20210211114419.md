## Installazione


su MYSQL:
CREATE DATABASE `paperlit` /*!40100 DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_0900_ai_ci */ /*!80016 DEFAULT ENCRYPTION='N' */;



composer install
php artisan migrate:fresh
php artisan breeze:install
sudo n stable
npm install && npm run dev
